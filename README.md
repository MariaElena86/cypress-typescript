E2E Testing – Registro de Usuario con Cypress

Se implementaron pruebas end-to-end utilizando Cypress para validar el flujo de registro de usuarios. Los tests cubren escenarios como el registro mediante número de teléfono y la verificación del envío de código, así como el registro por email. Se utilizan comandos personalizados para mejorar la reutilización y mantener el código limpio y mantenible.
